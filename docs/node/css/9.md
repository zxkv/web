```css

	margin: 0;
	padding: 0;
}

	width: 100%;
	height: 100%;
}

html {
	/* 灰色模式 */
	/* -webkit-filter: grayscale(1);
				filter: grayscale(1); */
}

#app {
	width: 100vw;
	height: 100vh;
}

#app h1 {
	line-height: 100px;
	font-size: 40px;
	font-weight: bold;
	text-align: center;
}

h2 {
	line-height: 100px;
```

置灰效果关键代码：

```css
html {
	/* 灰色模式 */
	-webkit-filter: grayscale(100%);
	filter: grayscale(100%);
	filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1);
}
```

主要用到的就是  **CSS3**样式的 **filter（滤镜）**属性
