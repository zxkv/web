# 编辑器配置 - Cursor

### IDE 简介

&emsp;&emsp;`Cursor` Built to make you extraordinarily productive, Cursor is the best way to code with AI.「Cursor 打造让您极具生产力的产品，Cursor 是使用 AI 进行编码的最佳方式。」

&emsp;&emsp;`Cursor` 同 `Windsurf Editor`一样，也可以直接采用 **vscode** 的配置，按照 **vscode** 的需求配置安装插件即可，无需额外安装，前提需要注册账号才可以使用编辑器。默认安装插件的源受网络限制，下载和搜索插件比较慢，如果要求高，需要下载插件到本地进行安装，或者自行配置代理来加速插件的下载和安装。

### 对比分析

| 功能         | Cursor AI Editor                       | VSCode                                     |
| ------------ | -------------------------------------- | ------------------------------------------ |
| **AI 集成**  | 内置 AI，开箱即用，体验流畅            | 依赖插件（如 Copilot），需要额外配置       |
| **插件生态** | 插件生态较弱                           | 插件丰富，支持几乎所有主流工具、框架和语言 |
| **可定制性** | 功能固定，可定制性较低                 | 高度可定制，支持修改设置和安装插件         |
| **性能**     | 小型项目性能良好，但可能不适合大型项目 | 对大部分项目都表现良好，性能稳定           |
| **离线能力** | 核心 AI 功能需要网络                   | 完全支持离线使用                           |
| **适用场景** | 快速原型开发、小型项目、初学者         | 复杂项目、多语言支持、大型团队合作         |
