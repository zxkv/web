# 编辑器配置 - Windsurf

### IDE 简介

&emsp;&emsp;`Windsurf Editor` 是由 **Codeium** 推出的一款「代理式」 **AI 编程工具**，内置了 **Codeium** 的所有功能，包括语言、插件、模板、代码片段等。支持`GPT 4o`、`Claude 3.5 Sonnet`和`Cascode Base`AI 工具。无需代理，即可直接切换 AI 工具使用，便于开发过程中代码智能提示和问题查找。

&emsp;&emsp;`Windsurf Editor` 可以直接采用 **vscode** 的配置，按照 **vscode** 的需求配置安装插件即可，无需额外安装，前提需要注册账号才可以使用编辑器。默认安装插件的源受网络限制，下载和搜索插件比较慢，如果要求高，需要下载插件到本地进行安装，或者自行配置代理来加速插件的下载和安装。

### 插件源配置

- 配置入口：首选项 -> 设置 -> 用户 -> [**搜索配置项名称**] -> [**填写配置项值**]

- 配置项 1：Windsurf : Marketplace Extension Gallery Service URL
- 配置值：`https://marketplace.visualstudio.com/items`

- 配置项 2：Windsurf : Marketplace Gallery Item URL
- 配置值：`https://marketplace.visualstudio.com/items`

::: danger ⚠️ 注意事项:

- **windsurf** 插件源于 **vscode** 插件源对比相对来说会少一些，如果所需插件搜索不到，即可按照上述方式替换现有编辑器的插件源

:::

### 适用场景

| **适用人群**             | **Windsurf**                               | **VSCode**                                 |
| ------------------------ | ------------------------------------------ | ------------------------------------------ |
| **初学者**               | 更适合，可能提供简化的功能和更低的学习成本 | 对初学者友好，但可能需要学习插件配置       |
| **轻量化用户**           | 非常适合快速编辑或小型项目                 | 相对重量级，但也支持轻量化配置             |
| **高级开发者**           | 可能功能不足，不适合复杂项目               | 丰富的功能和插件支持，适合高级用户         |
| **AI 辅助需求**          | 如果集成 AI 功能，可能表现更佳             | 需要安装插件（如 Copilot）才能实现 AI 辅助 |
| **多语言开发**           | 可能支持有限                               | 支持几乎所有主流编程语言                   |
| **团队协作和复杂工作流** | 可能不够强大                               | 非常适合，支持 Git、远程开发和复杂项目管理 |

### 对比分析

| 功能           | **Windsurf**                               | **VSCode**                       |
| -------------- | ------------------------------------------ | -------------------------------- |
| **启动速度**   | 可能更快，轻量化设计                       | 启动速度较快，但可能受插件影响   |
| **插件生态**   | 插件生态可能较弱                           | 插件市场非常丰富                 |
| **功能全面性** | 更适合特定场景，可能不够全面               | 功能全面，支持各种开发需求       |
| **社区支持**   | 可能较弱，问题解决依赖官方或开发者自行探索 | 社区庞大，有丰富的资源和支持     |
| **适用场景**   | 小型项目、快速开发、AI 辅助                | 大型项目、复杂工作流、多语言支持 |

&emsp;&emsp;如果 Windsurf 是一款新兴的编辑器，它可能会在特定领域（如 AI 驱动的开发）表现出色，但由于生态系统和功能的局限性，可能暂时无法取代 VSCode 的通用性和广泛支持。
