# 前端开发记录

#### 简介

&emsp;&emsp;本模块意在记录开发过程中遇到的问题、解决方案以及查阅资料寻求解决方案过程中发现和遇到的一些软件和工具。闲暇之余，整理分析，做归类总结，方便后续开发工作中查阅资料寻求解决方案。

#### 依赖

&emsp;&emsp;面对不同的需求和业务场景，需要不同的工具和软件，本模块主要记录一些开发过程中使用到的一些不错的 NPM 依赖包，以及自己分析和整理总结相关的 NPM 模块包，后续闲暇之余也会整理分析，做归类总结，更新时间不定。

#### 软件

&emsp;&emsp;软件模块主要记录一些开发过程中使用到的一些不错的软件，后续会对对应软件作相应的使用以及使用心得做记录和总结。若有好的软件推荐和分享，欢迎 📮 邮件告知：zxkang@88.com ，或者前往 [我的掘金博客](https://juejin.cn/user/184373682901309/posts) 私信 or 留言。
