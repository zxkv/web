#### 辅助软件

-   **ONE-KEY-HiDPI**：[https://github.com/xzhih/one-key-hidpi](https://github.com/xzhih/one-key-hidpi)
    -   「🔥」一键开启 macOS HiDPI，适配屏幕显示器，优化显示器显示模糊问题
    -   「💻」Mac
-   **Motrix**：[https://motrix.app/zh-CN](https://motrix.app/zh-CN)
    -   「🔥」一款全能的下载工具(支持下载 HTTP、FTP、BT、磁力链接等资源)
    -   「💻」Mac、Win、Linux

::: danger 注意事项：

-   [**one-key-hidpi**](https://github.com/xzhih/one-key-hidpi/blob/master/README-zh.md)**：macOS** 的 **DPI** 机制和 **Windows** 下不一样，比如 **1080p** 的屏幕在 **Windows** 下有 125%、150% 这样的缩放选项，而同样的屏幕在 **macOS** 下，缩放选项里只是单纯的调节分辨率，这就使得在默认分辨率下字体和 UI 看起来很小，降低分辨率又显得模糊。通过此工具，可以一键开启 **macOS** **HiDPI**，适配屏幕显示器，优化显示器显示模糊问题。

:::
