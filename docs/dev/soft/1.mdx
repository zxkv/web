#### 辅助软件

-   **ONE-KEY-HiDPI**：[https://github.com/xzhih/one-key-hidpi](https://github.com/xzhih/one-key-hidpi)

    -   「🔥」一键开启 Mac OS HiDPI，适配屏幕显示器，优化显示器显示模糊问题
    -   「💻」Mac OS

-   **Motrix**：[https://motrix.app/zh-CN](https://motrix.app/zh-CN)

    -   「🔥」一款全能的下载工具(支持下载 HTTP、FTP、BT、磁力链接等资源)
    -   「💻」Mac OS、Windows、Linux

-   **Gopeed**：[https://gopeed.com/zh-CN](https://gopeed.com/zh-CN)

    -   「🔥」一款现代化的下载器(开源、轻量、原生，支持（HTTP、BitTorrent、Magnet 等）协议下载)
    -   「💻」Mac OS、Windows、Linux、Android、iOS

::: danger 注意事项：

-   [**one-key-hidpi**](https://github.com/xzhih/one-key-hidpi/blob/master/README-zh.md)**：Mac OS** 的 **DPI** 机制和 **Windowsdows** 下不一样，比如 **1080p** 的屏幕在 **Windowsdows** 下有 125%、150% 这样的缩放选项，而同样的屏幕在 **Mac OS** 下，缩放选项里只是单纯的调节分辨率，这就使得在默认分辨率下字体和 UI 看起来很小，降低分辨率又显得模糊。通过此工具，可以一键开启 **Mac OS** **HiDPI**，适配屏幕显示器，优化显示器显示模糊问题。

:::
